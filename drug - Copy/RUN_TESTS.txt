===============================================
         TROUBLESHOOTING
===============================================

Error: "Server is not running"
------------------------------
Solution: Start the server in the first window
cd "drug - Copy"
node server.js


Error: "Cannot find module"
----------------------------
Solution: Install dependencies
cd "drug - Copy"
npm install


Tests Failed?
-------------
Check if:
1. Server is running (window 1)
2. Port 3000 is available
3. Databases exist (verification.db, tracking.db)
